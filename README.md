Talize is a finance manager app for tracking personal assets, understanding financial position, and keeping money decisions organized in one place.

## Features

- Secure account access with Google sign-in.
- Personal asset tracking foundation for savings, investments, and other holdings.
- Dashboard structure for viewing balances, portfolio movement, and financial summaries.
- Support for light and dark themes.
- Starter structure for extending the app with accounts, goals, budgets, liabilities, and investment insights.

## Setup

Install Flutter, then clone the repository and install dependencies:

```bash
flutter pub get
```

## Run

Run the app on a connected device or simulator:

```bash
flutter run
```

Run the analyzer:

```bash
flutter analyze
```

Run tests:

```bash
flutter test
```

## Build

Build an Android APK:

```bash
flutter build apk
```

Build an Android App Bundle:

```bash
flutter build appbundle
```

Build for iOS:

```bash
flutter build ios
```

Build for web:

```bash
flutter build web
```
